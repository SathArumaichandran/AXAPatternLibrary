Modal box

  • A modal box is a dialogue box, used to obstruct and get immediate action from users such as save a quote, etc
  • Positioned center and top with10px minimum margin for mobile, 20px minimum margin for tablet and 30px minimum margin for desktop
  • Modal box opened on top of 60% black faded overlay
  • On scroll, close link on white bar will become sticky i.e. float top
  • At AXA, we use Source sans pro 18px for copy, about 60 characters takes up 460px width and with 60px padding either side for desktop and tablet landscape which make modal box max-width 580px with outer margin on either side 50px for desktop
  • For mobile and tablet potrait, 20px padding and 20px outer margin

Accessibility

On the first land on the modal box, the screen reader should read the title or copy of the modal box first instead of reading the close link. Tab should be switch between elements within the modal box not outside.

There are few options to close the modal box, Click on "close" link, outside the modal box, pressing the escape key or enter key when "close" is in focus. Close of the modal box returns and focus the element which triggered the modal box.

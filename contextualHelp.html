Contextual help

  • Contextual help is a popup tip (sister of tooltip) used to provide additional information when users click/tap of more info link
  • Always show on screen if users need necessary information to complete the task and only accommodate the additional information in contextual help or popup tip
  • Keep the content short
  • Opened popup tip dismissed on click of the other element, close icon or outside the popup tip. (Only for desktop)
  • Positioned to not to obstruct on other elements
  • "More info" link label turned to "Close info" while the contextual help opened

Accessibility

On the first land on the contextual help, the screen reader should read the title of the contextual help first instead of reading the close link. Tab should be switch between elements within the contextual help not outside.

There are few options to close the contextual help, Click on "close" link, outside the contextual help, pressing the escape key or enter key when "close" is in focus. Close of the contextual help and focus the element which triggered the contextual help. https://www.w3.org/TR/wai-aria-practices-1.1/examples/dialog-modal/dialog.html

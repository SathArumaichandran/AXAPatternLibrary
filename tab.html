Tab

  • Tabs are used to switch content within the same view
  • Tabs are parallel in nature. Display the relevant content to the selected tab
  • Place tabs on top of the switching content
  • One of the tabs UI is always pre-selected. Tab option label should be short
  • Mobile tab options are transformed to anchor links with optional "content" title and all tab content shown by default

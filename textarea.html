Text area

  • Textarea is to let users enter more than single line entry however this comes with 1000 character limitation to let users to enter a message which can be short, conveyed to our back-office staff and data storage. In most cases, it is an optional field
  • Also, nice to have a feature which let users make the textarea wider by dragging the bottom right. I think this feature comes by default for <textarea> HTML tag
  • Scrollbar and expand item of the element are browser default styling

Loading feedback/indicator

  • Loading feedback is to inform users that the system is processing their request
  • If page takes longer than two seconds, it is better to inform the users the loading state
  • Use looped animation if it takes only few seconds (about 6 seconds) to load or use percent done animation if it takes to load 6+ seconds
